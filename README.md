# Backend web-chat service

### Description
Pet project created to learn symfony basics and technology interesting for me

## Stack

- Php 8.5
- Symfony 8.1
- Redis
- Postgres 18
- Nginx

## Local deploy

Create .env files from .env.example files in each directory where .env.example exists. You can use default values or setup your own

After .env files were created use 

```bash
make up
```
